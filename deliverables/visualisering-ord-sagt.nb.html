I et skuespil lader vi scene vre den tidsmssige udvikling. S vi tller hvad hver rolle siger for hver scene.

Hvor mange dialoger er der pr. scene?

Tilfj scene indeks

Da vi nsker at visualisere, hvor meget der tales over tid, har vi brug for et tal, der reprsenterer denne tid. Dette tal lader vi vre et indeks for hver scene, som vi beregner i det flgende.

Frst skal vi have beregnet antallet af scener i hver akt. Det gres med flgende kode.

my_play %>% 
   group_by(act_number)  %>% 
   summarise(scenes = max(scene_number)) %>% 
   pull(scenes) %>% cumsum() -> act_length

act_length

act_length er nu en liste, hvor element i angiver hvor mange scener der har vret til og med akt i. Fx er det tredje element i listen 28, hvilket betyder, at der i akt 1 til 3 sammenlagt er 28 scener. number_of_scenes er det samlede antal scener.

Vi kan nu lave en funktion, som givet et aktnummer og scenenummer kan beregne den givne scenes nummer i sekvensen af alle scener

calc_scene_index <- function(act_number, scene_number, act_length) {
   ifelse(is.numeric(scene_number),
      scene_number + ifelse(act_number==1, 0, act_length[act_number-1]),
      NA
   )
}

Vi bruger den som flger til at finde fx hvilket indeks den 4. scene i akt 3 har

calc_scene_index(3, 4, act_length)
calc_scene_index(2, 6, act_length)

Visualiser

For at visualisere hvem der siger hvor meget i de enkelte scener, skal vi erstatte det sagte med hvor mange ord, der bliver sagt. Dette gr vi ved at benytte funktionen str_count(spoke, '\\w+') som tager spoke og tller hvor mange ord, der er i spoke. Denne vrdi tilfjer vi i en ny kolonne, som vi kalder n_spoken_words.

(my_play %>% 
  mutate(n_spoken_words = str_count(spoke, '\\w+')) %>% 
  select(n_spoken_words, everything()) -> tmp)

Vi er nu ved at vre klar til at visualisere, hvem der siger hvor meget i fx Barselsstuen. Vi mangler bare at kombinere ovenstende kode

my_play %>% 
  # add a scene index for the x-axis
  rowwise() %>% 
  mutate(scene_index = calc_scene_index(act_number, scene_number, act_length)) %>% 
  # remove rows that are not dialogue
  filter(act != "", scene != "", !is.na(spoke)) %>%
  
  # select only the columns, that we are interested in
  select(scene_index, act_number, scene_number, speaker, spoke) %>% 
  
  # add the number of spoken words
  mutate(n_spoken_words = str_count(spoke, '\\w+')) %>% 
  # remove the spoken words
  select(-spoke) %>%
  
  # group the play in scene_index and speaker, ignoring the test
  group_by(scene_index, act_number, scene_number, speaker) %>% 

  # Ensure that each speaker only appears once in each scene, i.e. sum the words spoken by each speaker
  # Lastly store the new data frame in a new reference for later use
  summarise(words = sum(n_spoken_words), act_number, scene_number) %>%
  distinct() -> my_summary

Vi kan nu bruge to R funktioner: if_else() og %in%. Den frste har tre argumenter. Det frste er et logisk udtryk. Hvis dette udtryk er sandt, returneres resultatet af det andet argument. Hvis udtrykket er falsk, returneres resultatet af det tredje argument. Fx

if_else(2==2, "Korrekt", "FAKE NEWS")
if_else("Trump"=="Klog", "Korrekt", "FAKE NEWS")

Vi vil ogs benytte funktionen %in%, som tjekker for om det der str til venstre er indeholdt i den liste der str til hjre. Fx

12 %in% c(1,2,3,4)
12 %in% c(1, 2, 3, 12, 24)

Med de to funktioner, kan vi nu skrive denne kode Husk at disse ndringer er lavet til Barselstuen

my_summary %>% 
  ungroup() %>% 
  
  # count all versions of Leander as one
  mutate(
    speaker = if_else(speaker %in% c("Leander", "Leander p Kn"), "Leander", speaker)) %>%
  
  # count all versions of Barselskvinden as one
  mutate(
    speaker = if_else(speaker %in% c("Barselskonen", "Barselskvinden", "Barselsqvinde"), "Barselskvinden", speaker)) %>% 
  
  # Count all versions of kllingen as one
  mutate(
    speaker = if_else(speaker %in% c("Klling", "Kllingen"), "Kllingen", speaker)) %>% 
  
  ggplot(aes(fill = speaker, y = words, x = scene_index)) +
    geom_bar(stat="identity", width = 50) +
